# 第一章: 悪しき構造の弊害を知覚する

第一章では、良い構造を知るためには、悪い構造とはどんなもので、どのような弊害があるかを見ていきます。

このようにプログラミングでは、ベストプラクティスやシルバーバレットのようなものを求めるのではなく、アンチパターンを学習することで悪い構造にならないように努めていくことが求められます。
